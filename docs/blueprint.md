# **App Name**: SkillPathAI

## Core Features:

- User Profile: Capture user's skills, courses, projects, and achievements.
- Skill Assessment: Algorithm to analyze skills and identify gaps.
- Recommendation Engine: Suggest courses/projects based on career goals using AI to reason with external data as a tool.
- Dashboard Visualization: Visualize skill progression and career pathways.
- External API Integration: Connect with LinkedIn or course platforms.
- Sector-Specific Skill Frameworks: Focus on healthcare, agriculture, and urban sector skills.

## Style Guidelines:

- Background color: Dark navy (#0f172a) to create a modern and sophisticated feel.
- Card background: #111827, slightly lighter than the background for contrast and depth.
- Primary accent: Sky blue (#38bdf8) for interactive elements and key actions, conveying trust and innovation.
- Secondary accent: Green (#22c55e) to highlight success, growth, and positive outcomes.
- Text primary: Off-white (#e5e7eb) for readability against the dark background.
- Text secondary: Gray (#9ca3af) for less important information, maintaining visual hierarchy.
- Border: Dark gray (#1f2937) for subtle separation of UI elements.
- Body and headline font: 'Inter', a sans-serif font known for its readability and modern aesthetic, ensuring clarity across all UI elements.
- Good spacing to allow content to breathe, improving overall user experience.
- Rounded corners to soften the UI, making it more approachable and less clinical.
